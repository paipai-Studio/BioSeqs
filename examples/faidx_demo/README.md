# FASTA Index Demo

Demonstrates pyfaidx-style FASTA indexing:
- Building .fai index from FASTA content
- Random access to subsequences
- Full sequence retrieval
- Sequence length lookup
- Index existence check
- .fai format serialization and parsing
- Compatible with samtools faidx

## Run

```bash
moon run examples/faidx_demo/main.mbt
```
