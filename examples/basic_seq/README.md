# Basic Sequence Operations

Demonstrates fundamental sequence operations including:
- DNA/RNA/Protein creation
- Complement and reverse complement
- Transcription and back-transcription
- Translation with stop codon handling
- Sequence search and count
- Sequence manipulation (upper/lower/reverse/slice)
- GC content and molecular weight calculation
- Melting temperature (Tm) calculation
- Checksums (CRC32, GCG, SEGUID)

## Run

```bash
moon run examples/basic_seq/main.mbt
```
