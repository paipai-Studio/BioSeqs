# SAM/VCF Demo

Demonstrates high-throughput sequencing data parsing:
- SAM alignment file parsing
- Sequence dictionary access
- Alignment record details (flag, position, CIGAR)
- VCF variant file parsing
- Variant properties (SNP, insertion, deletion)
- Genotype access for samples

## Run

```bash
moon run examples/sam_vcf_demo/main.mbt
```
